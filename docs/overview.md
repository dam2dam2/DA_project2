# 서울시 공공자전거 대여 데이터 분석 프로젝트 설명

2015년부터 2025년까지의 조각난 따릉이 대여 데이터를 통합하여 분석하는 프로젝트입니다.

## 파일 구조

- `src/collect_data.py`: 데이터 수집 및 통합 스크립트
- `src/dashboard.py`: Streamlit 대시보드
- `data/daily_bike_rentals_2015_2025.csv`: 통합된 데이터셋
- `docs/overview.md`: 본 문서

## 데이터 수처 및 통합 과정

1. `DA_project2/data` 폴더에서 엑셀 및 CSV 파일들을 수집합니다.
2. 각 파일의 '대여일자'와 '대여건수' 컬럼을 찾아 표준화합니다.
3. 중복된 날짜의 데이터를 합산하여 하나의 시계열 데이터로 변환합니다.

## 대시보드 사용법

```bash
# 가상환경 활성화 후 (또는 직접 경로 지정)
streamlit run src/dashboard.py
```

- 좌측 사이드바에서 분석할 연도를 선택할 수 있습니다.
- 연도별, 월별, 요일별 이용 트렌드를 차트로 확인할 수 있습니다.
